Question Answer
What is an object? An object is an instance of a class. Think of a class as a factory that makes objects.
What does the initialize method do? The initialize method is called whenever a new object based on the class is created.
What is the scope of local, instance, class, and global variables? Local variables can only be accessed in a given method.
Instance variables are also known as object variables (i.e. @method) and have scope within the current object.
Class variables (i.e. @@method) can be accessed within the class.
Global variables (i.e. $x) can be accessed anywhere, including in classes and objects.
