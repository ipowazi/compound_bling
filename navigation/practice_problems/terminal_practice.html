Identify the three parts of the following command: $ ls -l ~/Desktop ls is a utility. Sometimes utilities can be run without flags or arguments.
-l is a flag that alters how the utility operates. In this case, the -l flag will cause more information to be presented.
~/Desktop is an argument to the utility. The number of arguments depends on the utility (some take none, one, or many)

When you run $ ls with no flag, the output does not distinguish between files and folders. What flag can be used, so the $ ls output distinguishes between files and folders? $ ls -F (the folders will have slashes at the end)
Run $ ls and output the hidden files as well (and have the output distinguish between files and folders) $ ls .
What are two hidden folders in every folder and what do they do? . and .. are hidden in every folder. The . link takes you back to the same folder than you are currently in. The .. link takes you back up to the parent folder.
