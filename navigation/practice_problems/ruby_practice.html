What is the difference between if and unless?

The code in an if block will execute if the condition is true. The code in an unless block will execute if the condition is false.

Ruby's philosophy is that there should be many ways to solve the same problem. Other languages, like Python, think that there should only be one way (or only one best way) to solve a problem (the Pythonic way). Rails seems to be much more opinionated than Ruby.
