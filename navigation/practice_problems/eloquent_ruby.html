What is the shortcut way to write the following hash? names = {:first => "matt", :second => "bill"} names = {first: "matt", second: "bill"}
Write a function that will sum an arbitrary amount of numbers. def sum(*nums)
 nums.inject{|total, number| total + number}
end
Write a method that prints "person's favorite authors are ...add list of favorite authors..." def fav_authors(person, *authors)
 puts "#{person}'s favorite authors are:"
 authors.each {|author| print author + " "}
end
Use the following method to sum this array: arr = [1, 2, 3]
def add(num1, num2, num3)
 num1 + num2 + num3
end 		add(*arr)
Write two other ways that this function can be called:
def print_name(name_hash)
 puts "#{name_hash[:first]} #{name_hash[:last]}"
end

p print_name({first: "cool", last: "dude"}) 		p print_name(first: "cool", last: "dude")
p print_name first: "cool", last: "dude"
Convert h = {:sport=>"soccer", :place=>"europe", :why=>"dunno"} to [[:sport, "soccer"], [:place, "europe"], [:why, "dunno"]] h.map {|x| x}
Write a method to calculate the average word length in an array def average_word_length(array)
 total = array.inject(0.0) {|result, word| word.size + result}
 total / array.length
end
Create an array with 10 nil elements and the string "cool" in the 11th index position. arr = []
arr[10] = "cool"

This is something that is not desirable (especially if you put the string cool at position 1000 and created an array with 1,000 nils), so watch for this.
